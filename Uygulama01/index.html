Uygulamalı Web Geliştirme

Sıfırdan ileri seviyeye kadar gelişim

Dersin Amacı

"Uygulamalı Olarak Web Geliştirme" yaparak sıfırdan ileri seviyeye kadar web geliştirme ve web tasarım dünyasına profesyonel olarak adım atmak.

Hedefim

Kurs sonunda bir web geliştirici için gereken tüm araçları ve konuları öğrenmiş olarak iş hayatıma başlamak.

Kurs sonunda her sektöre hitap eden tanıtım siteleri , blog tarzındaki web stelerini kolayca hazırlayabilmek. Hatta bu tasarlamdığım web stelerini profesyonel biçimde yönetim paneli hazırlamak.
